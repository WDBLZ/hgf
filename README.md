## 求职招聘系统

作者：亚双义、一真

时间：2020年4月6日~2020年4月7日

实现功能：三个用户：企业、求职者、平台管理员；支持在线查看简历、投递简历、下载简历等

实现技术：LayUI+asp.net MVC+SqlServer+echarts

平台：Visual Studio 2017
